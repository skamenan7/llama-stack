# Distributions Overview

A distribution is a pre-packaged set of OGX components that can be deployed together. Pick a distribution that matches your deployment target, or build your own.

:::tip
Not sure which distribution to use? Start with the **[Available Distributions](/docs/distributions/list_of_distributions)** page to compare options side by side.
:::
