# API Providers

Llama Stack lets you swap out different implementations for the same API. Providers come in two types:

- **Remote**: adapts an external service (Ollama, OpenAI, vLLM, Bedrock, etc.)
- **Inline**: runs in-process within Llama Stack (FAISS, sentence-transformers, Llama Guard, etc.)

At least one inline provider exists for each API so you can run a fully featured stack locally.
