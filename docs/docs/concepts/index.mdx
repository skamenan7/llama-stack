Llama Stack is a service-oriented platform. A few concepts arise from this design which may not feel natural if you're coming from other LLM frameworks.

- **[Architecture](architecture)** - how the system is structured, request flow, provider dispatch
- **[APIs](apis/)** - the core APIs and their stability levels
- **[Distributions](/docs/distributions/)** - pre-configured deployment packages
- **[Resources](resources)** - models, shields, vector stores and their lifecycle
